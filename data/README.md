# PiFocus data directory

This folder keeps header-only CSV files in git so the expected local data schemas are visible.

During a live run, PiFocus writes rows into these files on the local machine:

- `live_camera_features.csv`: features extracted from camera frames.
- `live_camera_predictions.csv`: visual attention predictions generated from camera features.
- `activity_log.csv`: laptop app/window/idle context.
- `fused_focus_log.csv`: fused output consumed by the dashboard.
- `session_features_v2.csv`: optional training dataset using the same camera feature schema.

The committed CSV files contain headers only. Local runtime rows can be regenerated by running the pipeline.
